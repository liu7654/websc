# 🛍️ Web 商城 - 现代化电商前端项目

[![Vue 3](https://img.shields.io/badge/Vue-3.5.10-4FC08D?logo=vue.js)](https://vuejs.org/)
[![Vite](https://img.shields.io/badge/Vite-5.4.0-646CFF?logo=vite)](https://vitejs.dev/)
[![Pinia](https://img.shields.io/badge/Pinia-2.1.7-FFD859)](https://pinia.vuejs.org/)

一个基于 **Vue 3 + Vite + Pinia** 的现代化电商前端项目，具备完整的用户、商品、购物车、订单管理功能，并经过全面的 UX 优化。

## ✨ 项目特色

- 🎨 **精美的 UI 设计** - 现代化的界面设计，流畅的动画效果
- 📱 **完全响应式** - 完美适配桌面端和移动端
- ⚡ **极致性能** - 图片懒加载、搜索防抖、骨架屏加载
- ♿ **无障碍访问** - 完善的 ARIA 标签和键盘导航
- 🔐 **用户认证** - 完整的登录/注册/权限管理
- 🎯 **用户体验** - Toast 提示、表单验证、确认对话框、页面过渡动画

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖

```bash
# 克隆项目
git clone <your-repository-url>

# 进入项目目录
cd websc

# 安装依赖
npm install
```

### 开发模式

```bash
# 启动开发服务器（默认端口 5173）
npm run dev
```

开发服务器启动后，访问 http://localhost:5173 即可查看项目。

### 生产构建

```bash
# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 七、开发注意事项
1. **Mock 数据** - 使用 axios-mock-adapter 拦截请求，生成假 token
2. **API 封装** - 所有请求统一通过 api 目录的模块发起
3. **状态管理** - Pinia + localStorage 双存储，刷新不丢失
4. **路由守卫** - 未登录自动跳转登录页，已登录无法访问登录/注册页
5. **数据持久化** - 购物车、登录状态保存到 localStorage
6. **精度计算** - 金额计算使用 toFixed(2) 保留两位小数
7. **组件化开发** - 高度模块化，可复用组件提取到 components 目录
8. **组合式函数** - 复用逻辑封装为 composables
9. **响应式优先** - 移动端优先设计，向上兼容桌面端
10. **性能优化** - 懒加载、防抖、骨架屏等性能优化措施

## 九、📝 开发规范

### 代码风格
- 使用 Vue 3 Composition API
- 组件使用 `<script setup>` 语法
- CSS 使用 scoped 样式
- 遵循 Vue 官方风格指南

### 文件命名
- 组件文件：PascalCase（如 `ProductCard.vue`）
- 工具文件：camelCase（如 `useToast.js`）
- 样式文件：kebab-case（如 `main.css`）
